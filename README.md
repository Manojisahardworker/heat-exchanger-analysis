# Heat Exchanger Dataset Analysis

## 📌 Overview
This project analyzes a large dataset of heat exchanger performance using **Python (NumPy, Pandas, Matplotlib)**.  
It demonstrates **data cleaning, statistical analysis, and regression modeling** in an engineering context.

## ⚡ Features
- Forward fill for missing values
- Calculation of average temperature drop, standard deviation of flow rate
- Identification of maximum drop intervals
- Regression analysis (NumPy polyfit) of flow rate vs. temperature drop
- Visualizations: line plots, histograms, scatter plots

## 🛠️ Requirements
- Python 3.8+
- NumPy
- Pandas
- Matplotlib
